📚 Caesar Cipher – How It Works

Encryption Formula

E(x) = (x + n) mod 26

where x = letter position (A=0…Z=25), n = shift value

1

Convert to Number

Use ord(letter) − 65 to get position (A=0, B=1 … Z=25)

2

Add Shift Value

Add the shift n. Example: A + 3 → position 3

3

Apply Modulo 26

Use mod 26 so Z wraps around back to A

4

Convert Back

Use chr(position + 65) to get the encrypted letter
